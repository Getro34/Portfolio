Process Automation

  • 2021 / 2023
  • Excel VBA

While I was looking after the maintenance of the outside of the houses, I had the opportunity to see their system for managing entrances and exits. So I suggested doing an Excel spreadsheet using VBA to keep track of all the tenants in each house over several years. Firstly, I had to consider the case where a tenant would return several times in the same year.

In the same style, during an internship I had to automate the list of pupils in a school, but here it was even more complicated because I had to take several things into account:
- change of id if repeating a year
- Standardisation of surnames and first names for spaces, hyphens etc.
- check whether pupils have been expelled / changed school
- create function for random password with x characters
- link with google admin database
